Privacy Policy

Last updated: March 12, 2026

1. Information We Collect

Meme Bot does not collect personal data from end users. The tool operates solely on behalf of the account owner to automate content posting.

2. TikTok API Data

Meme Bot uses the TikTok Content Posting API. Access tokens are stored securely as encrypted GitHub Secrets and are never shared with third parties.

3. Data Storage

No user data is stored on external servers. Video content is sourced from the account owner's private AWS S3 bucket.

4. Third-Party Services

Meme Bot interacts with the following third-party services:

  • TikTok API (content posting)
  • Instagram Graph API (content posting)
  • Amazon S3 (video storage)
  • MongoDB (video metadata)

5. Data Retention

No personal data is retained by Meme Bot.

6. Contact

For privacy-related questions, contact us via the GitHub repository.
